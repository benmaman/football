Upload your CSV file

Please ensure your CSV file is formatted with the following columns:

  • player: The name or ID of the player.
  • score: The score or ranking of the player.
  • goalkeeper: Indicate if the player is a goalkeeper with '1' or leave empty if not.

The file should include headers for each column. Here's an example:

        player,score,Goalkeeper
        John Doe,15,1
        Jane Smith,12,
        Bob Johnson,10,
